# Star Shapes

A 2D arcade style game with a space theme.

The game is [avaiable to play on this URL](https://star-shapes.herokuapp.com/ "Star Shapes URL").

### Tech
- Elm Language
- Elm SocketIO
- HTML 5 Canvas
- Node JS
- Express
- Socket IO
- Bacon JS
